# Highest Posterior Density intervals for objects of class `mtar`

Highest Posterior Density intervals for objects of class `mtar`

## Usage

``` r
# S3 method for class 'mtar'
HPDinterval(obj, prob = 0.95, ...)
```

## Arguments

- obj:

  an object of class `mtar` generated by a call to the function
  [`mtar()`](https://lhvanegasp.github.io/mtarm/reference/mtar.md).

- prob:

  a numeric scalar in the interval \\(0,1)\\ giving the target
  probability content of the intervals. By default, `prob` is set to
  `0.95`.

- ...:

  Optional additional arguments for methods. None are used at present.

## See also

[`HPDinterval`](https://rdrr.io/pkg/coda/man/HPDinterval.html)
